# LiveLook Teacher Console

Enterprise React frontend for an AI classroom monitoring platform. The current build uses mocked live data and a websocket-ready service layer so backend integration can attach without reshaping the UI.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Stack

React 19, Vite, TypeScript, TailwindCSS, Framer Motion, React Router, TanStack Query, Zustand, React Hook Form, Zod, Recharts, Sonner, Lucide React.
